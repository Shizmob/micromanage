micromanage
===========

An AFK streamer, IRC bot and metadata updater.

Usage
-----
Edit `micromanage/config.py` and start it by executing `python micromanage`.

Requirements
------------
* Python 2.x
  - Twisted
  - pyOpenSSL
  - lxml with `etree`
  - BeautifulSoup
  - pylibshout
  - tagpy
